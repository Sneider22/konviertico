# Konvierte - Ecosistema Financiero Digital (Venezuela) 🇻🇪🚀

Konvierte es una infraestructura de datos financieros gratuita y de código abierto diseñada para democratizar el acceso a cotizaciones de divisas y criptoactivos en tiempo real. 🔋🔌💎

## 🚀 Características Principales
*   **Redundancia de Datos**: Extracción multi-fuente sincronizada con BCV e Instagram oficial.
*   **Filtrado Inteligente**: Algoritmo de filtrado de mercado P2P para evitar ruidos de cotización erróneos.
*   **Arquitectura Edge**: Microservicios desplegados en la infraestructura global de Vercel para latencia ultra-baja.
*   **Open Source**: Código abierto disponible para integración en aplicaciones móviles, ERPs y plataformas e-commerce.

## 🛠️ Stack Tecnológico
*   **Frontend**: React + TypeScript + Vite + Tailwind CSS.
*   **Backend (API)**: Serverless Functions (Vercel Core).
*   **Base de Datos**: Supabase (PostgreSQL) para persistencia e histórico.
*   **Diseño**: UX/UI minimalista enfocado en la productividad del desarrollador.
